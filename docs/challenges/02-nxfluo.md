# Challenge 2 — Convert to NXfluo and Identify Missing Information

## Goal
Convert the provided experimental dataset into a NeXus file that follows the **NXfluo application definition**. In the same NXentry, include an **NXnote** that documents missing information in the dataset and any limitations in representing the experiment using this application definition. The final file must be uploaded to NOMAD.

## What you get
- A directory containing raw experimental data files  
- A master metadata file with scan parameters  
- A minimal ELN YAML file with partial metadata  
- A mapping template (in JSON format) showing how fields can be mapped to NeXus structure (incomplete and must be adapted)
